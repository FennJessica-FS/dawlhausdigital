# Dawlhaus Digital | Jessica "Dawl" Fenn

A high-performance, neon-infused digital hub for small businesses, makers, and creators.

## 🚀 The Vision
Specialized digital solutions for those who need more than just a pretty page. Built with a "Proof of Concept" approach for restaurants, personal brands, and event activations.

## 🛠️ Tech Stack
* **Languages:** HTML5, CSS3 (Glassmorphism UI), JavaScript
* **Deployment:** GitHub Pages
* **Forms:** Formspree Integration

## 🎨 Design Language
* **Theme:** Dark Mode / Neon Accents
* **Typography:** Bungee & Space Grotesk
* **Features:** Noise overlays, scanline effects, and interactive cursor glows.
